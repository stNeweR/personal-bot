# Personal bot

## Что может этот бот
1) Помогает сфокусироваться на работе благодаря помодоро таймеру
2) Помогает вести учёт расходов
3) Напоминает вам читать интернет ресурсы которые вы уже добавили

## Реализованный функцонал
### Помодоро таймер
- [ ] Добавление конфига помодоро сессии для пользователя
- [ ] Запуск помодоро сессии 
- [ ] Отправка уведомлений об окончании како-ого либо периода сессии
- [ ] Вывод статистики помодоро за неделю, день, месяц
- [ ] Добавление цели на сессию и получение результата этой сессии
- [ ] Остановка помодоро таймера

### Учёт расходов
- [ ] Добавление категории расходов
- [ ] Добавление трат, с описанием того, что купилил 
- [ ] Общие сводки по тратам за день, неделю, месяц
- [ ] Отображение расходов в гугл таблице (или простой таблице в mini app)
- [ ] Удаление/изменение расходов или трат
- [ ] Возможность добавить ограничение на каждую категорию в месяц и сравнение с прошлыми

### Работа с заметками 
- [ ] Сохранение заметок из телеграм каналов (перессылка)
- [ ] Добавление категорий для сортировки заметок/ресурсов
- [ ] Можно ли реализовать сохранение аудио или фото сообщений

### Чтение статей
- [ ] Добавление статей для чтения 
- [ ] Выбрать график в какой период будет обновляться статья 
- [ ] Добавление рецензии на статью

### Внутренние задачи 
- [ ] Написать микросервис уведомлений на Golang (и настроить общение с основным сервисом)
- [ ] Добавить в проект Kafka и Redis
- [ ] Настроить деплой телеграм бота
- [ ] Сделать Telegram Mini App с применением Vue, Pinia, Tailwind CSS (может быть ещё и с Nuxt)
