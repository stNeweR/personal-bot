# Review Code Skill

Этот навык переключает conversation на агента **@reviewer** для выполнения задач по ревью кода.

## Когда использовать

Используйте этот навык, когда пользователь просит:

1. **Ревью кода** — «Сделай ревью кода», «Проверь этот файл»
2. **Анализ архитектуры** — «Проверь архитектуру проекта», «Есть ли нарушения Clean Architecture»
3. **SOLID проверка** — «Проверь на SOLID», «Найди нарушения принципов SOLID»
4. **Security аудит** — «Найди уязвимости», «Проверь на безопасность»
5. **Анализ зависимостей** — «Проверь зависимости между модулями»

## Как переключить на @reviewer

1. **Подтвердите задачу** — Убедитесь, что пользователю нужно именно ревью кода (а не написание нового кода)

2. **Переключите агента** — Используйте инструмент `task` с subagent_type `general-purpose` для запуска @reviewer:

```yaml
task:
  description: "Code review"
  subagent_type: "general-purpose"
  prompt: |
    Выполни ревью кода как агент @reviewer.
    
    Задача пользователя: {описание задачи}
    Файл/директория: {путь если указан}
    
    Используй только разрешённые инструменты:
    - read, glob, grep_search, run_shell_command
    
    Примени навыки:
    - clean-architecture
    - solid  
    - security-review
    
    Не модифицируй файлы, только анализируй.
    Сформируй отчёт по формату из reviewer.md
```

3. **Дождитесь результата** — Агент @reviewer выполнит анализ и вернёт отчёт

## Ограничения @reviewer

- **Только анализ** — не редактирует и не создаёт файлы
- **Без тестов** — не запускает тесты и не выполняет код
- **Без внешних API** — использует только локальные файлы
- **Глубина рекурсии** — до 5 уровней зависимостей

## Формат отчёта

@reviewer возвращает отчёт в формате:

```markdown
# Code Review Report

## Обзор
- Проверено файлов: X
- Найдено проблем: Y
- Критических: Z

## Clean Architecture
...

## SOLID
...

## Security
...

## Рекомендации по приоритетам
...
```

## Примеры использования

**Пользователь:** «Сделай ревью директории src/»

**Вы:** Переключаете на @reviewer с задачей анализа директории

**Пользователь:** «Проверь файл UserService.php на SOLID»

**Вы:** Переключаете на @reviewer с указанием конкретного файла

**Пользователь:** «Найди уязвимости в controllers/»

**Вы:** Переключаете на @reviewer с фокусом на security
